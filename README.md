# LinguaBuddy Translator

## Project Description
LinguaBuddy Translator is a multilingual web application developed using HTML, CSS, and JavaScript.  
It enables users to translate text between multiple languages, including English, Hindi, Kannada, Malayalam, Tamil, Telugu, French, and Spanish.  
The project integrates a translation API to process input text and display the translated output.  
Additional features include a copy button and text-to-speech functionality for enhanced usability.  
The interface is designed with a modern gradient theme to provide a simple and user-friendly experience.

## Aim
To design and implement a smart, user-friendly web-based language translator that supports multiple languages and improves accessibility for users.

## Algorithm
1. Accept input text from the user.  
2. Select source and target languages from dropdown menus.  
3. Send the text to the translation API.  
4. Receive and display the translated output.  
5. Provide options to copy or speak the translated text.

## Output Summary
The application successfully translates text between selected languages.  
It provides a clean interface with gradient background, copy and speak buttons, and accurate translation results.  

## Conclusion
LinguaBuddy Translator demonstrates the integration of web technologies with APIs to create a functional and creative multilingual tool.  
It is simple, effective, and ideal for beginner-level web development projects.
